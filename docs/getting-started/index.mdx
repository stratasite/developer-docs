# Getting Started

Welcome to Strata! This guide will help you get started with building semantic analytics models using the Strata platform.

## What is Strata?

Strata is a semantic layer platform that transforms raw database tables into business-ready analytics models. It enables self-service analytics by providing a business-friendly interface to your data while maintaining the power and flexibility of SQL.

**Key Features:**
- **Semantic Modeling**: Define business-friendly fields and metrics
- **Query Optimization**: Automatic SQL generation with cost-based routing
- **Multi-Datasource**: Query across multiple databases seamlessly
- **Self-Service**: Business users explore data without writing SQL
- **Version Control**: Track changes with migrations and Git
