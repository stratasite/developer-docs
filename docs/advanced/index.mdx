# Advanced Features

Power user features for complex semantic modeling scenarios.

## Overview

Advanced features enable sophisticated analytics capabilities beyond basic dimensions and measures.

## Features

- **[Snapshot Measures](/docs/advanced/snapshot-measures)** - Point-in-time analysis
- **[Exclusions](/docs/advanced/exclusions)** - Filter-based measure exclusions
- **[Inclusions](/docs/advanced/inclusions)** - Multi-level calculations
- **[Partitions](/docs/advanced/partitions)** - Data availability constraints
- **[Cost Optimization](/docs/advanced/cost-optimization)** - Query routing and performance
- **[Decorators](/docs/advanced/decorators/temporal)** - Transformations (temporal, window, contribution)
- **[Multi-Datasource](/docs/advanced/multi-datasource)** - Cross-datasource queries

## When to Use

Use advanced features when you need:
- **Point-in-time analysis** - Snapshot measures
- **Complex filtering** - Exclusions
- **Multi-level calculations** - Inclusions
- **Data constraints** - Partitions
- **Performance tuning** - Cost optimization
- **Time-based analysis** - Temporal decorators
- **Cross-database queries** - Multi-datasource
